NEWSLETTER SUBSCRIPTION API

Endpoint: POST /api/users/newsletter-subscribe
Access: Public (no authentication required)

Request Body:
{
  "email": "user@example.com",        // Required
  "firstName": "John",                 // Optional
  "lastName": "Doe"                    // Optional
}

Success Response (200):
{
  "success": true,
  "message": "Successfully subscribed to newsletter"
}

Error Responses:

400 - Bad Request:
{
  "message": "Email is required"
}
OR
{
  "message": "Invalid email format"
}

500 - Server Error:
{
  "message": "Failed to subscribe to newsletter. Please try again later."
}

Notes:
- Email is validated for format
- Contact is added to Brevo list 5 (newsletter list)
- firstName and lastName are optional but will be stored if provided
- Email is automatically lowercased and trimmed
